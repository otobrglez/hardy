# hardy

Hardy is a [tttm][tttm] challenge server written by [Oto Brglez][ob] in Rust.

## Development && Docker

1. Use `devenv` like a pr0.
2. `docker build -t otobrglez/hardy .`
3. `docker run --name=hardy --rm -p 8884:8884 otobrglez/hardy`
